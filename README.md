## Dependências

###### Sanity

Instalação:
```npm install -g @sanity/cli```
```sanit init```

###### Sanity Client

Instalação:
 ```yarn add @sanity/client```

###### Sanity Image-url

Instalação:
 ```yarn add @sanity/image-url```


## Iniciando o servidor local
Primeiro devemos iniciar o Sanity e depois o servidor de desenvolvimento do next.js

###### Sanity:
```bash
npm run dev
#ou
npx sanity dev
```

###### Next.js:
```bash
npm run dev
# or
yarn dev
```
